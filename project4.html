Hellen Lassen

Igennem en bekendt kom jeg i kontakt med kunsteren Hellen Lassen, der gerne ville have udarbejdet et interaktivt galleri, hvor hun kunne vise sine værker frem.

I samarbejde med hende, planlagte vi, hvordan siden skulle se ud, og så kodet jeg siden ved brug af HTML, CSS og vanilla JavaScript.

Siden er dynamisk og gøre brug af en Wordpress Rest API database.
